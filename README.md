# Pex Challenge

This is a solution the a coding challenge.

 - assumes output result order does not matter

### How To Run

Java 8+ is needed

Compile and run
 - reads data from input.txt in same dir
 - outputs to output.csv (will append)

```sh
$ javac ImageWorker.java MainChallenge.java
$ java MainChallenge
```
